# Users Tree Table

Это репозиторий с решением тестового задания: SPA веб-приложение для отображения иерархической таблицы пользователей с возможностью сортировки, фильтрации и раскрытия вложенных структур.

Проект выполнен с акцентом на производительность, чистоту кода, строгую типизацию и доступность (a11y), без использования готовых UI-библиотек компонентов.

## Архитектурные решения
### Построение дерева

Исходный плоский массив преобразуется в иерархию на основе id и parentId.
Порядок элементов во входных данных не предполагается отсортированным.

### Сортировка

Поддерживается сортировка по:
- balance (с предварительным парсингом строки в число)
- email

Сортировка применяется рекурсивно ко всем уровням дерева

### Фильтрация

- Фильтр по isActive
- Реализован рекурсивно для всей структуры

## Технологический стек

*   **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
*   **Core**: [React 19](https://react.dev/) & React DOM 19
*   **Language**: TypeScript 5 (Strict Mode)
*   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (без использования компонентных библиотек вроде MUI или AntD)
*   **Linting/Formatting**: ESLint 9, Prettier

## Как запустить проект

### 1. Клонировать репозиторий
```bash
git clone https://github.com/shinokamix/shinokamix-users-tree-table.git
cd shinokamix-users-tree-table
```

### 2. Установить зависимости
```bash
npm install
```
### 4. Запустить проект в режиме разработки
```bash
npm run dev
```

### 5. Открыть приложение в браузере

http://localhost:3000
